## 生成順序

`outer_shell.py` は現在、次の順序で `外殻` を作ります。

1. XY 面に `外殻` スケッチを作成する
2. 内殻の XY 断面ラインをスケッチへ投影し、通常のスケッチ線へ変換する
3. `B-I-C-D-E-J` 側の外周線と投影済み内殻アウトラインでできる外側領域を選択する
4. 選択領域を `-Z` 方向へ `3.0 mm` 押し出して `外殻` ボディを作成する
5. `内殻` ボディをツールにして `Cut` し、内側の干渉部分を除去する
6. `内殻` と接する `外殻` 側の面群を `-0.2 mm` オフセットし、接触逃がしを作る
7. 基準点 `(-5.0, -35.0, 0.0) mm` を通る `外殻` ボディ上 XY 面を `+Z` 方向へ `29.0 mm` 追加押し出しする
8. 基準点 `(-5.0, -35.0, -3.0) mm` を通る `外殻` ボディ上 XY 面を `-Z` 方向へ `0.2 mm` 追加押し出しする
9. 原点を通る XY 平面に `外殻外周円` スケッチを作成する
10. `z = -3.2 mm` の XY 平面に `外殻底面外側` スケッチを作成する
11. `J-F-G-H-C-D` 側の閉領域を `-Z` 方向へ `2.8 mm` `Join` 押し出しする
12. `外殻蓋部斜面内部` で外殻を分割し、上側ボディを削除する
13. `z = -6.0 mm` の XY 平面上で、指定円弧エッジへ `R3.0` フィレットをかける
14. `外殻外周円` の内側領域を `+Z` 方向へ `40 mm` `Cut` 押し出しする
15. 円切り抜きで `z = 0 mm` 面にできた円弧エッジへ `R6.0` フィレットをかける
16. `外殻` を蓋部斜面内部平面で分割した後、`Y = 35.0 mm` の `外殻基準構造` 上 XZ 面に `外殻Lボタン開口部` スケッチを追加する
17. H-I-J-K の 4 点を結ぶ矩形内の閉領域だけを選び、`-Y` 方向へ `20 mm` `Cut` 押し出しする
18. 切り取り後に、J 点を通る `Y` 軸方向直線エッジへ `R8.0` フィレットをかける
19. 続けて、I 点を通る `Y` 軸方向直線エッジへ `R4.0` フィレットをかける
20. `x = -59.8 mm` の YZ オフセット平面に `外殻Lボタン開口部内部仕様` スケッチを追加する
21. `AEBCFD` の半円状閉領域を選び、軸 `AD` を中心に `360 deg` 回転 `Cut` する

## 外殻本体の仕様

### 基本スケッチ

- スケッチ名: `外殻`
- スケッチ面: XY 面
- 基準点: `A(-5.0, -23.0, 0.0)`、`B(-5.0, -35.0, 0.0)`、`C(-90.0, -35.0, 0.0)`、`D(-90.0, 35.0, 0.0)`、`E(-51.8, 35.0, 0.0)`
- 補助点: `I(-5.0, -20.0, 0.0)`、`J` は `x=-51.8` 上で内殻上面傾斜線と一致する点
- 使用線分: `BI`、`BC`、`CD`、`DE`、`EJ`

### 本体押し出し

- 押し出し対象: 外周線と内殻断面投影で区切られた外側領域
- プロファイル選択基準: `(-80.0, 0.0, 0.0)` mm 近傍
- 押し出し方向: `-Z`
- 押し出し量: `3.0 mm`
- 押し出し方法: `NewBody`

### 内殻との接触逃がし

- 対象: `外殻` と `内殻` が接する `外殻` 側面群
- 処理: `Offset Faces`
- オフセット量: `-0.2 mm`
- 面名: `外殻内殻接面`

### 外周上面の追加押し出し

- 対象面: `(-5.0, -35.0, 0.0) mm` を通る `外殻` ボディ上の XY 面
- 押し出し方向: `+Z`
- 押し出し量: `29.0 mm`
- 押し出し方法: `Join`

## 底面外側の追加仕様

### 外側面の追加押し出し

- 対象面: `(-5.0, -35.0, -3.0) mm` を通る `外殻` ボディ上の XY 面
- 押し出し方向: `-Z`
- 押し出し量: `0.2 mm`
- 押し出し方法: `Join`

### 底面外側スケッチ

- スケッチ名: `外殻底面外側`
- スケッチ面: `z = -3.2 mm` の XY 平面
- A円:
  中心 `(0.0, 5.0, -3.2)` mm
  通過点 `(-5.0, -23.3, -3.2)` mm
- B直線:
  `(-25.0, 30.0, -3.2)` mm から `(-25.0, 14.0, -3.2)` mm を通る鉛直線
- C直線:
  `(-25.0, 30.0, -3.2)` mm から `(-51.8, 28.0, -3.2)` mm へ向かう直線
- D直線:
  `(-51.8, 35.0, -3.2)` mm から `(-51.8, 28.0, -3.2)` mm を通る鉛直線
- 追加点:
  `E` は A円と B直線の交点
  `F(-5.0, -35.0, -3.2)` mm
  `G(-90.0, -35.0, -3.2)` mm
  `H(-90.0, 35.0, -3.2)` mm
  `I(-5.0, -34.0, -3.2)` mm
  `J` は `x=-5.0` 上で A円と `F-I` 延長線が交わる下側交点
- 追加線:
  `FG`
  `GH`
  `HD`
  `FJ`

### 底面外側領域の押し出し

- 押し出し対象: `J-F-G-H-C-D` 側の閉領域
- プロファイル選択基準: `(-60.0, 0.0, -3.2)` mm 近傍
- 押し出し方向: `-Z`
- 押し出し量: `2.8 mm`
- 押し出し方法: `Join`

### 底面外側フィレット

- 対象平面: `z = -6.0 mm` の XY 平面
- 対象円弧: `(-5.0, -23.3, -6.0)` mm と `(-25.0, -19.174, -6.0)` mm 近傍を通る円弧エッジ
- フィレット半径: `R3.0`

## 蓋部側の分割

- 基準面: `内殻` の `内殻蓋部斜面`
- 作成平面名: `外殻蓋部斜面内部`
- 作成方法: `内殻蓋部斜面` を法線方向へ `0.2 mm` オフセットした平行平面
- 分割処理: `外殻` をこの平面で分割する
- 削除対象: 分割後の上側ボディ

## 外周円の切り抜き

### 外周円スケッチ

- スケッチ名: `外殻外周円`
- スケッチ面: `(0.0, 0.0, 0.0) mm` を通る XY 平面
- 円の中心点: `(-42.0, 5.0, 0.0)` mm
- 円の半径: `R33`

### 外周円の切り抜き

- 押し出し対象: `外殻外周円` の内側領域
- 押し出し方向: `+Z`
- 押し出し量: `40 mm`
- 押し出し方法: `Cut`
- 参加ボディ: `外殻` のみ
- 補足: 処理中は `内殻` を一時退避して、切り抜き対象から外す

### 外周円切り抜き後のフィレット

- 対象: `z = 0 mm` の XY 面にある、押し上げ切り抜きでできた円弧エッジ
- フィレット半径: `R6.0`

## 補足

- `外殻` は `inner_shell.py` が先に生成した `内殻` ボディを参照する前提です。
- 外殻底面外側のスケッチは確認用要素も含みます。Fusion 上ではプロファイルの閉じ方を目視確認してください。
- `1.py` は現在の全体確認用で、既存の `z = 0 mm` 側開口押し出しと `Y = 35.0 mm` 側の L ボタン開口部切り取り・フィレットまでを含みます。
- `2.py` は `Y = 35.0 mm` 側の `外殻Lボタン開口部` に加えて、`x = -59.8 mm` 側の `外殻Lボタン開口部内部仕様` スケッチと回転切り取りの確認用として使います。
