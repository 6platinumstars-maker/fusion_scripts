## 生成順序

`outer_shell.py` は現在、次の順序で `外殻` を作ります。

1. XY 面に `外殻` スケッチを作成する
2. 内殻の XY 断面ラインをスケッチへ投影し、通常のスケッチ線へ変換する
3. `B-I-C-D-E-J` 側の外周線と投影済み内殻アウトラインでできる外側領域を選択する
4. 選択領域を `-Z` 方向へ `3.0 mm` 押し出して `外殻` ボディを作成する
5. `内殻` ボディをツールにして `Cut` し、内側の干渉部分を除去する
6. `内殻` と接する `外殻` 側の面群を `-0.2 mm` オフセットし、接触逃がしを作る
7. 基準点 `(-5.0, -35.0, 0.0) mm` を通る `外殻` ボディ上 XY 面を `+Z` 方向へ `29.0 mm` 追加押し出しする
8. 基準点 `(-5.0, -35.0, -3.0) mm` を通る `外殻` ボディ上 XY 面を `-Z` 方向へ `0.2 mm` 追加押し出しする
9. 原点を通る XY 平面に `外殻外周円` スケッチを作成する
10. `z = -3.2 mm` の XY 平面に `外殻底面外側` スケッチを作成する
11. `J-F-G-H-C-D` 側の閉領域を `-Z` 方向へ `2.8 mm` `Join` 押し出しする
12. `外殻蓋部斜面内部` で外殻を分割し、上側ボディを削除する
13. `z = -6.0 mm` の XY 平面上で、指定円弧エッジへ `R3.0` フィレットをかける
14. `外殻外周円` の内側領域を `+Z` 方向へ `40 mm` `Cut` 押し出しする
15. 円切り抜きで `z = 0 mm` 面にできた円弧エッジへ `R6.0` フィレットをかける
16. `x = -51.8 mm` の YZ オフセット平面に `外殻端修正` スケッチを追加し、拡張矩形領域を `-X` 方向へ `3.0 mm` `Cut` 押し出しする
17. `x = -51.8 mm` の YZ オフセット平面に `外殻YZ矩形切り取り` スケッチを追加し、E-F-G-H の矩形領域を `-X` 方向へ `6.0 mm` `Cut` 押し出しする
18. `x = -5.0 mm` の YZ オフセット平面に `外殻側面押し出し` スケッチを追加し、E-F-G-H の矩形領域を `+X` 方向へ `5.0 mm` `Join` 押し出しする
19. `x = -9.6 mm` の YZ オフセット平面に `外殻はめ込み微調整` スケッチを追加し、指定矩形領域を `-X` 方向へ `0.2 mm` `Cut` 押し出しする
20. `外殻蓋部斜面隙間部` の主スケッチと、コントローラ抑え用の追加スケッチを作成する
21. `外殻` を蓋部斜面内部平面で分割した後、`Y = 35.0 mm` の `外殻基準構造` 上 XZ 面全体を `-Y` 方向へ `2.8 mm` `Cut` 押し出しする
22. `外殻Lボタン開口部斜面切り取り` スケッチの閉領域を両側へ `0.2 mm` `Cut` 押し出しする
23. `x = -59.8 mm` の YZ オフセット平面に `外殻Lボタン開口部内部仕様` スケッチを追加する
24. `AEBCFD` の半円状閉領域を選び、軸 `AD` を中心に `360 deg` 回転 `Cut` する
25. `外殻蓋部斜面隙間部` の主領域を `+Z` 方向へ `3.0 mm` `Join` スイープし、最上面を `外殻蓋部斜面外側` と命名する
26. コントローラ抑えを、`外殻蓋部斜面隙間部 追加` の領域から斜面法線負方向へ `3.0 mm` `Join` 押し出しし、終端の `円弧NM` と `PN` / `MF` に `R3.0` フィレットをかける
27. `外殻下部受止` を追加するため、YZ 面上の `QRSTUVW` 閉領域を `-X` 方向へ `5.0 mm` `NewBody` 押し出しする
28. 新規ボディの `Q-Q'` と `R-R'` に対応する `X` 方向エッジへ `R3.0` フィレットを適用する
29. フィレット後断面を `x = -5.0 mm` 平面で取得し、その断面領域と `ABCD` 領域をロフトして新規ボディを作る
30. `BHIJ` 領域と `BJU'V'` 領域をロフトして新規ボディを作る
31. 上記 3 つの新規ボディを外殻本体へ `Combine Join` する
32. `蓋部装飾切取り①` として、`x = -41.895 mm` 平面上 `ABC` 領域を軸 `OP` まわりに `+150° / -120°` 回転 `Cut` する
33. `蓋部装飾切取り②` として、`外殻蓋部斜面外側` 上の `FG` 基準 90° 傾斜平面で `HIJ` 領域を作り、`-X 6.111 mm / +X 5.356 mm` `Cut` 押し出しする
34. `蓋部装飾切取り③` として、ボディ上平面 `A'B'C'` と `A'B'H'` を取得し、その 2 面の間を `Cut` ロフトする
35. `蓋部装飾切取り④` として、`MNQ` 領域を軸 `KL` まわりに `-120°` 回転 `Cut` し、続けて同領域を法線方向へ `20.0 mm` と反対方向 `1.0 mm` `Cut` 押し出しする
36. `蓋部装飾切取り⑤` として、`XYZ` 領域を法線方向へ `20.0 mm`、反対方向へ `1.0 mm` `Cut` 押し出しする
37. `外殻下部受止` の追加後、`z = -3.2 mm` の XY 平面に `外殻はめ込み修正` スケッチを作成する
38. `OR`、`RP`、円弧 `PQ`、`QO` で囲まれる閉領域を `+Z 25.0 mm` `Cut` 押し出しする
39. 切り取りで新たにできた `A面` と、`外殻下部受止` 側の既存 `B面` を `Cut` ロフトする
40. 上記切り取りで分離した小ボディを削除し、最大体積の外殻ボディを残す
41. `外殻はめ込み修正` の追加後、`z = -3.2 mm` の XY 平面に `外殻底部外側斜面` スケッチを作成する
42. `AH-短い側の円弧HF'-F'G-CD-DE-EA` で囲まれる閉領域を `-Z 2.8 mm` `Join` 押し出しする
43. `内殻` ボディを一時退避したうえで、`IJLK` 閉領域から `+Y 5.0 mm` の切取り用ボディを作り、`Combine Cut` で外殻本体を切り取る
44. 同様に、`JLMO` 閉領域から `+Y 70.0 mm` の切取り用ボディを作り、`Combine Cut` で外殻本体を切り取る

## 外殻本体の仕様

### 基本スケッチ

- スケッチ名: `外殻`
- スケッチ面: XY 面
- 基準点: `A(-5.0, -23.0, 0.0)`、`B(-5.0, -35.0, 0.0)`、`C(-90.0, -35.0, 0.0)`、`D(-90.0, 35.0, 0.0)`、`E(-51.8, 35.0, 0.0)`
- 補助点: `I(-5.0, -20.0, 0.0)`、`J` は `x=-51.8` 上で内殻上面傾斜線と一致する点
- 使用線分: `BI`、`BC`、`CD`、`DE`、`EJ`

### 本体押し出し

- 押し出し対象: 外周線と内殻断面投影で区切られた外側領域
- プロファイル選択基準: `(-80.0, 0.0, 0.0)` mm 近傍
- 押し出し方向: `-Z`
- 押し出し量: `3.0 mm`
- 押し出し方法: `NewBody`

### 内殻との接触逃がし

- 対象: `外殻` と `内殻` が接する `外殻` 側面群
- 処理: `Offset Faces`
- オフセット量: `-0.2 mm`
- 面名: `外殻内殻接面`

### 外周上面の追加押し出し

- 対象面: `(-5.0, -35.0, 0.0) mm` を通る `外殻` ボディ上の XY 面
- 押し出し方向: `+Z`
- 押し出し量: `29.0 mm`
- 押し出し方法: `Join`
