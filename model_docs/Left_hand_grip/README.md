# Left_hand_grip README

`fusion_app/Left_hand_grip.py` 用のモデル説明です。

## 対象ファイル

- スクリプト: [fusion_app/Left_hand_grip.py](/home/ps/fusion_scripts/fusion_app/Left_hand_grip.py)
- 実行用コピー: [1.py](/home/ps/fusion_scripts/1.py)

## 現在の構成

`Left_hand_grip.py` は現在、次の順序で形状を作ります。

1. `底面内部` スケッチからベースボディを作成する
2. `分割断面` スケッチから三角形断面を作成する
3. 三角形断面を `-X` 方向へ 10 mm `Join` 押し出しし、できた斜面に `底面斜面` という名前を付ける
4. YZ 面に `上部止部` スケッチを作成する
5. `上部止部` を `-X` 方向へ 25 mm 押し出して別ボディとして作成する
6. `上部止部` のうち、最も Z が高い XY 面を `+Z` 方向へ 3 mm 押し上げる
7. YZ 面に `下部止部` スケッチを作成する
8. `下部止部` を `-X` 方向へ 10 mm `Join` 押し出しする
9. `底面斜面` と `下部止部` が交差してできる `-X` 方向エッジに `R5.0` フィレットをかける
10. `下部止部平面` `下部止部曲面` `下部止部曲面基準点` の名前を付ける

## 上部止部の確定仕様

- 名称: `上部止部`
- スケッチ面: YZ 面
- 押し出し方向: `-X`
- 押し出し量: 25 mm
- 外側円弧半径: `R9.0`
- 内側円弧半径: `R6.15`
- 内側円弧開始点: `(0.00, 30.00, 0.00)` mm
- 外側基準点: `(0.00, 30.00, -3.00)` mm
- 上面の追加押し上げ: 最上位 XY 面を `+Z` へ 3 mm

上端の寸法は、先端に三角突起が出ないよう、外側半径と内側半径の差に合わせて調整しています。

## 底面斜面の確定仕様

- スケッチ面: `分割断面` を作成した `x = 0.0` の面
- スケッチ内容: `分割断面` スケッチから三角形断面を作成する
- 押し出し方向: `-X`
- 押し出し量: `10 mm`
- 押し出し方法: 既存ベースへ `Join`
- 名称: `底面斜面`

`底面斜面` は、三角形断面の押し出しで新たにできる傾斜平面です。後続の `下部止部` フィレットでは、この面と `下部止部` が共有する `-X` 方向エッジを使います。

## 下部止部の確定仕様

- 名称: `下部止部`
- スケッチ面: YZ 面
- スケッチ基準点: `(0.00, -20.00, 0.00)` mm
- 対角基準点: `(0.00, -30.00, 7.00)` mm
- 押し出し方向: `-X`
- 押し出し量: `10 mm`
- 押し出し方法: 既存ベースへ `Join`
- フィレット対象: `底面斜面` と `下部止部` が交差して作る `-X` 方向エッジ
- フィレット半径: `R5.0`
- 面名称:
  `下部止部曲面`
  フィレット後にできる円筒曲面です。
  `下部止部平面`
  押し出し後の上側 XY 平面です。
- 点名称:
  `下部止部曲面基準点`
  フィレットを行った軸と `分割断面` の交点です。

## 名前を付けた面・辺・頂点の説明

現在のコードで明示的に名前を付けているのは、スケッチ、面、ボディ、頂点です。

### スケッチ

- `底面内部`
  XY 平面上に作るベース外形スケッチです。
- `分割断面`
  `x = 0.0` の面上に作る三角形の断面スケッチです。
- `上部止部`
  YZ 面上に作る J 字断面のスケッチです。
- `下部止部`
  YZ 面上に作る長方形断面のスケッチです。

### 面

- `分割断面`
  `create_split_triangle_on_face()` 内で属性 `fusion_scripts:name=分割断面` を付けている面です。ベースボディのうち `x = 0.0` にある側面を指します。
- `底面斜面`
  `分割断面` の三角形を `-X` へ 10 mm `Join` 押し出ししたときにできる傾斜平面です。
- `下部止部平面`
  `下部止部` の押し出しでできる上側の XY 平面です。
- `下部止部曲面`
  `底面斜面` と `下部止部` の共有エッジへ `R5.0` フィレットを適用してできる曲面です。

### ボディ

- `上部止部`
  YZ 面スケッチから生成する J 字形状のボディです。別ボディとして作成したあと、最上位 XY 面をさらに `+Z` へ 3 mm 押し上げています。

### 辺

現在のコードでは、辺に対して固有名は付けていません。

ただし `下部止部` の追加後は、`底面斜面` と `y = -20.00 mm` の `下部止部` 側面が共有する `-X` 方向エッジを探索して `R5.0` フィレット対象にしています。

探索に使える関数:

- `find_longest_edge(face)`
- `find_shortest_edge(face)`
- `find_edge_by_constant_axis(face, axis, value, tolerance=1e-6)`

今後、特定の辺に属性名を付けたい場合は `add_named_attribute()` を辺エンティティにも使えます。

### 頂点

現在のコードでは、次の頂点に名前を付けています。

- `下部止部曲面基準点`
  `フィレットを行った軸` と `分割断面` の交点です。`x = 0.0` 側のフィレット対象エッジ端点を使っています。

探索に使える関数:

- `find_vertex_by_coordinates(vertices, x=None, y=None, z=None, tolerance=1e-6)`

今後必要になれば、頂点にも `add_named_attribute()` で名前を付けられます。
